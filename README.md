# TODO list console

## Descripción

El clasico todo list pero en consola y con persistencia local.

### Acciones disponibles

- `Crear tarea`
- `Listar tareas`
- `Listar tareas completas`
- `Listar tareas pendiente`
- `Completar tarea(s)`
- `Borrar tarea`
